# Position on Debate 002 — Workflow composition

**Agree.** 

A monolithic 200+ line YAML workflow file with deep nesting is impossible for LLM agents to edit without breaking syntax, and tedious for human review. Implementing a minimal, top-level `include:` list that merges fragments at load time provides necessary modularity for Phase 4 without inheriting the complexities of full YAML anchors.
