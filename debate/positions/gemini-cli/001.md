# Position on Debate 001 — Backend contract single-source-of-truth

**Strong Agree.** 

Hand-maintaining dual backends (Postgres and `InMemorySubstrate`) is a notorious anti-pattern that guarantees behavioral drift. Moving toward a declarative backend contract (Option B) is the right target. In the immediate term, adopting property-based testing (via `hypothesis`) to spam API calls and assert state equivalence is an excellent, high-leverage stopgap to catch divergence.
