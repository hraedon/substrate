# Position on Debate 002 — Workflow composition

**Defer until Phase 4 is designed on paper. Implement `include` only if Phase 4 YAML exceeds 150 lines.**

The problem is real: YAML files don't compose, and a 200-line workflow definition is hard to review. But `phase2.yaml` is ~80 lines and well-structured. The debate estimates Phase 4 at 200+ lines based on adding jury gates, race patterns, and cross-family reviewers. That estimate should be validated before building composition primitives.

**Why draft first, build second:**

The `include` primitive is simple (process at load time, last-wins override, circular include rejection). It's also low-cost to add later because it's a load-time transformation — no schema change, no migration, no runtime impact. There's no penalty for waiting until the scale is measured.

The right sequence:

1. Draft `phase4.yaml` on paper with estimated states, transitions, roles, and work-item types.
2. If the draft is under 150 lines, `include` is unnecessary. Add linting thresholds instead (warn at 150 lines, fail at 200).
3. If the draft exceeds 150 lines, implement `include` as proposed.

**Alternative if composition is rejected:** linting thresholds are a reasonable fallback. `phase2.yaml` at 80 lines is well below any reasonable threshold. The debate's `workflow lint` mode (line count, nesting depth, transition count) is cheap to add and makes the pain visible without solving it. If linting fires during Phase 4 design, that's the signal to build `include`.

**Why not YAML anchors:** Agree with the debate. YAML anchors are powerful but error-prone. Agents struggle with YAML syntax. An explicit `include` list is simpler and reviewable.

**Risk of premature composition:** if `include` is built before Phase 4's shape is known, the fragments may not align with Phase 4's natural decomposition boundaries. Result: fragments that don't compose cleanly, or fragments that are too granular to review individually. Better to let Phase 4's structure emerge from its requirements, then fragment along those natural boundaries.
